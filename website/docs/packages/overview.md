# Utility Packages

Standalone utility packages for modern web applications. Each package is independently versioned and published.

| Package                                         | Description                                                          |
| ----------------------------------------------- | -------------------------------------------------------------------- |
| [@ovineko/react-router](./react-router)         | Type-safe wrapper for React Router v7 with valibot schema validation |
| [@ovineko/clean-pkg-json](./clean-pkg-json)     | Zero-config tool to clean package.json before publishing             |
| [@ovineko/fastify-base](./fastify-base)         | Pre-configured Fastify server with observability integrations        |
| [@ovineko/datamitsu-config](./datamitsu-config) | Internal shared configuration for datamitsu tooling                  |
